# Ascent Foundations — Assignments

Per-session assignment decks for **Ascent Foundations**, the free digital-literacy bootcamp by [SwiftRoc](https://swiftroc.com/). Each deck covers what to build, how to submit, and how it's scored.

**Live (GitHub Pages):** `https://mrzeeshanniaz.github.io/ascent-bootcamp-assignments/`

- `index.html` — landing page linking everything

## How students use it
Open a deck, do the assignment in your own Google Drive sandbox, then submit your work link in your Ascent portal.

## Editing
These files are generated from the main Ascent repo (`scripts/gen_assignment_decks.mjs`). Edit there and re-run, then copy the built HTML here. Navigate decks with the arrow keys or the Prev/Next buttons.
